Available commands:

.add file
   creates a new password file
.clear
   clears the screen
.export path
   exports files to a new repository with a new password
.help
   prints this help
.html path
    exports to a password-protected html page
.lock
   locks the application
.lock ( pin | password | none )
   changes auto-lock type
.open path
   opens file by path
.pwd
   generates passwords
